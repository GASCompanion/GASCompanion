### As a Project Plugin

1. Download Plugin from Marketplace
2. Create a Plugin folder in your project
3. Copy the Plugin folder from Engine  to your project's plugin folder 
    - ex. `[Engine Directory]/Engine/Plugins/Marketplace/GASCompanion` to `[Project Directory]/Plugins/GASCompanion`
4. Start the project
